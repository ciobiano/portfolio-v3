## Introduction

Designed to simplify content management and administration, this CMS is an advanced and robust content management system. It provides a user-friendly interface and enables users to manage different parts of their digital content, such as inventory and billboards, with ease.

## Features

This CMS encompasses the following key entities:

- Billboards: Dynamic advertising spaces to grab user attention.
- Categories: Organizational units for content structuring.
- Sizes: Management of product sizes and variations.
- Manufacturers: Database of product manufacturers and suppliers.
- Inventory: Real-time tracking and control of stock levels.
- Orders: Comprehensive order management and processing.
- Settings: Customizable configurations to modify the CMS.

All of these ideas are pretty much fully implemented.

## API Routes

This CMS leverages API routes to enable efficient data communication. These routes facilitate the interaction between the front-end and back-end, ensuring the smooth operation of the system.

## Technologies

This Content Management System is fully built with [NextJs](https://Nextjs.org) and [React.js](https://reactjs.org).
It also uses [Tailwind CSS](https://tailwindcss.com) for the styling of the app, Tailwind CSS allows me to easily create amazing looking styles fast!
