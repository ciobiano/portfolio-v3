## Introduction

Eterna Care is a blood bank management system that helps to manage the blood bank and the donor information. It is a web-based application that can be used by blood banks to manage their blood bank and donor information. It is a user-friendly application that can be used by anyone with basic computer knowledge. It is a secure application that can be used to store sensitive information about donors and blood banks.

## Features

- User authentication and authorization
- Schedule appointments with laboratories
- View and manage appointments
- View and manage Blood-bank Inventory
- Responsive UI for a seamless user experience

## Full tech stack

- Frontend: React, Next.js, Tailwind CSS, Shadcn UI
- Backend: Node.js, Next.js Server Action
- Database: Prisma ORM with MongoDB
- State Management: React Query (TanStack Query)
- Validation: Zod, React Hook Form
